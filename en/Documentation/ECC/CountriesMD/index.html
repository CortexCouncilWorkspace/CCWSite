    CountriesMD

    Source SQL File - Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - (ecc) - CountriesMD

    Overview

    This view provides a comprehensive list of country-specific settings and configurations used in SAP systems. It combines data from the T005 and T005T tables to provide a single source of truth for country-related information.

    Data Sources

    Source Table/View Description
    T005 Country-specific settings and configurations
    T005T Country-specific texts and translations

    Key Use Cases

    • Retrieve country-specific settings for various business processes, such as address formatting, postal code validation, and tax calculations.
    • Obtain country-specific texts and translations for user interfaces, reports, and other documents.
    • Analyze country-specific data to identify trends, patterns, and potential areas for improvement.

    Query Logic

    The view combines data from the T005 and T005T tables using an INNER JOIN on the MANDT and LAND1 fields. This ensures that only country-specific settings and texts that match are included in the view.

    Possible relation with SAP Transactions

    Transaction Code Transaction Description
    OB01 Maintain Country-Specific Settings
    OB02 Maintain Country-Specific Texts
    OB03 Maintain Country-Specific Currencies
    OB04 Maintain Country-Specific Units of Measure
    OB05 Maintain Country-Specific Number Ranges
