    POSchedule

    Source SQL File - Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - (ecc) - POSchedule

    Overview

    This reporting view provides detailed information about schedule lines in SAP purchasing documents. It combines data from the EKET table with additional dimensions from the calendar_date_dim table, providing insights into the timing and status of schedule lines.

    Data Sources

    Source Table/View Description
    eket SAP table containing schedule line data for purchasing documents

    Key Use Cases

    • Analyzing the delivery schedule of purchasing documents
    • Monitoring the progress of goods receipts and deliveries
    • Identifying potential delays or issues in the supply chain
    • Reporting on purchasing performance and efficiency

    Query Logic

    The query joins the EKET table with the calendar_date_dim table on the EINDT and BEDAT fields to add temporal dimensions to the schedule line data. It also includes additional fields for budget information, handover details, and reminder messages.

    Possible relation with SAP Transactions

    Transaction Code Transaction Description
    ME23N Display Purchase Order
    ME22N Change Purchase Order
    ME21N Create Purchase Order
