    PlantsMD

    Source SQL File - Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - (ecc) - PlantsMD

    Overview

    This reporting view provides a comprehensive overview of plant-related data in SAP systems. It includes information such as plant name, address, key figures, and various indicators.

    Data Sources

    Source Table/View Description
    T001W Plant data table

    Key Use Cases

    • Analyzing plant-specific data
    • Reporting on plant performance
    • Identifying key plant characteristics

    Query Logic

    The view selects all relevant plant data from the T001W table.

    Possible relation with SAP Transactions

    Transaction Code Transaction Description
    MM01 Create Plant
    MM02 Change Plant
    MM03 Display Plant
