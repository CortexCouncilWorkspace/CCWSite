    Deliveries

    Archivo SQL de origen: Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - S4 - Deliveries

    Descripción general

    Esta vista proporciona datos de elaboración de informes para sistemas SAP. Combina datos de las tablas LIKP y LIPS para proporcionar una vista completa de la información relacionada con la entrega.

    Fuentes de datos

    Tabla/vista de origen Descripción
    LIKP Datos de cabecera para entregas
    LIPS Datos de ítem para entregas

    Casos de uso clave

    • Se puede utilizar para analizar patrones y tendencias de entrega
    • Se puede utilizar para identificar posibles áreas de mejora en el proceso de entrega
    • Se puede utilizar para rastrear el estado de entregas individuales

    Lógica de consulta

    La vista se crea utilizando una unión entre las tablas LIKP y LIPS. La unión se realiza en el campo VBELN, que es el número de entrega. Esto garantiza que todos los datos de una entrega determinada se incluyan en la vista.

    Posible relación con SAP Transactions

    Codigo de transacción Descripción de la transacción
    VL01N Crear entrega
    VL02N Cambiar entrega
    VL03N Mostrar entrega
