    InvoiceDocuments_Flow

    Archivo SQL de origen: Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - S4 - InvoiceDocuments_Flow

    Descripción general

    Esta vista de informes proporciona una descripción general completa de la información relacionada con el código de la empresa en los sistemas SAP. Incluye datos clave como el código de moneda, el código de la empresa, el texto de la empresa, el nombre de la ciudad, el país, el idioma, el plan de cuentas, la variante del año fiscal, la empresa, el área de control de crédito, el plan de cuentas del país y la gestión de fondos.

    Fuentes de datos

    Tabla/vista de origen Descripción
    t001 Tabla de Código de Empresa

    Casos de uso clave

    • Analizar configuraciones y ajustes específicos del código de la empresa
    • Informar sobre datos financieros por código de empresa
    • Identificar códigos de empresa asociados con unidades de negocio o ubicaciones específicas
    • Realizar análisis y conciliación de datos en múltiples códigos de empresa

    Lógica de consulta

    La vista selecciona varios campos de la tabla t001, que contiene información relacionada con el código de la empresa en los sistemas SAP.

    Posible relación con SAP Transactions

    Codigo de transacción Descripción de la transacción

    OB63 Definir Valores Predeterminados para Códigos de Empresa
    OB64 Definir Datos Adicionales para Códigos de Empresa
