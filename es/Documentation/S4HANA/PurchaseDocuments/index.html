    PurchaseDocuments

    Archivo SQL de origen: Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - S4 - PurchaseDocuments

    Descripción general

    Esta vista es una vista de informes para sistemas SAP que combina datos de las tablas EKPO y EKKO. Proporciona una vista completa de los datos de la orden de compra, incluidos los detalles a nivel de artículo, los detalles a nivel de encabezado e información adicional de las tablas relacionadas.

    Fuentes de datos

    Tabla/vista de origen Descripción
    ekpo Datos a nivel de artículo para órdenes de compra, que incluyen cantidad, precio e información de entrega.
    ekko Datos a nivel de encabezado para órdenes de compra, que incluyen información del proveedor, tipo de documento y condiciones de pago.

    Casos de uso clave

    • Se puede utilizar para analizar datos de órdenes de compra a nivel de artículo o encabezado.
    • Se puede utilizar para rastrear el estado de las órdenes de compra y sus artículos.
    • Se puede utilizar para identificar tendencias y patrones en el comportamiento de compra.

    Lógica de consulta

    La vista se crea utilizando una unión izquierda entre las tablas EKPO y EKKO. La unión se realiza en los campos MANDT y EBELN, que son el cliente y el número de orden de compra, respectivamente. También se incluyen campos adicionales de tablas relacionadas en la vista mediante uniones izquierdas.

    Posible relación con SAP Transactions

    Codigo de transacción Descripción de la transacción
    ME21N Crear orden de compra
    ME22N Cambiar orden de compra
    ME23N Mostrar orden de compra
    ME28 Liberar orden de compra
    ME29N Mostrar historial de orden de compra
