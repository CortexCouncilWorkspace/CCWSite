    ProductionOrders

    Arquivo SQL de origem - Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - S4 - ProductionOrders

    Visão geral

    Esta visão fornece uma visão geral abrangente das ordens de produção SAP e seus itens associados. Ela combina dados das tabelas AFKO, AFPO, AUFK, JEST e TJ02T para fornecer uma visão holística das informações relacionadas à ordem, incluindo status da ordem, detalhes de programação, requisitos de material e informações de custo.

    Fontes de Dados

    Tabela/Visão Descrição
    AFKO Cabeçalho da Ordem de Produção
    AFPO Item da Ordem de Produção
    AUFK Dados Mestres da Ordem
    JEST Status do Objeto
    TJ02T Textos de Status do Objeto

    Key Use Cases

    • Monitorar o status e o progresso da ordem de produção
    • Analisar o planejamento da ordem e o planejamento da capacidade
    • Rastrear os requisitos de material e a disponibilidade
    • Revisar estimativas de custo e custos reais
    • Identificar gargalos e áreas para melhoria

    Query Logic

    A visão combina dados de várias tabelas usando INNER JOINs para criar um conjunto de dados abrangente. A lógica da consulta garante que apenas dados relevantes sejam incluídos e que os relacionamentos entre as tabelas sejam mantidos.

    Possível relação com transações SAP

    Código da Transação Descrição da Transação
    CO03 Exibir Ordem de Produção
    CO02 Alterar Ordem de Produção
    CO11N Visão Geral da Ordem de Produção
    CO15 Processamento em Massa da Ordem de Produção
    MB51 Visão Geral do Estoque de Material
