    Arquivo SQL de origem - Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - S4 - ProfitCentersMD

    Visão geral

    Esta visão de relatório fornece uma visão geral abrangente dos dados do centro de lucro dos sistemas SAP. Ele combina dados das tabelas CEPC e CEPCT para fornecer uma única fonte de verdade para informações do centro de lucro.

    Fontes de Dados

    Tabela/Visão Descrição
    CEPC Dados Mestres do Centro de Lucro
    CEPCT Textos do Centro de Lucro

    Key Use Cases

    • Análise e relatórios do centro de lucro
    • Planejamento e orçamento financeiro
    • Gerenciamento de centro de custo
    • Preço de transferência
    • Contabilidade de joint venture

    Query Logic

    A visão filtra os dados das tabelas CEPC e CEPCT com base na data atual para garantir que apenas as informações mais atualizadas sejam exibidas. Ele também une as duas tabelas nos campos MANDT, PRCTR, DATBI e KOKRS para garantir que os dados sejam consistentes.
