    CostCenterAmountsHierarchy_SAMPLE

    Arquivo SQL de origem - Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - S4 - CostCenterAmountsHierarchy_SAMPLE

    Visão geral

    Esta visão de relatório fornece uma visão hierárquica dos centros de custo e seus documentos contábeis associados. Ele combina dados das tabelas SAP csks_hier, setheadert, CostCentersMD e AccountingDocuments para criar uma visão abrangente da hierarquia do centro de custo, descrições e dados financeiros.

    Fontes de Dados

    Tabela/Visão Descrição
    {{ project_id_src }}.{{ dataset_reporting_tgt }}.csks_hier Tabela de hierarquia SAP contendo informações de hierarquia de centro de custo.
    {{ project_id_src }}.{{ dataset_cdc_processed_s4 }}.setheadert Tabela SAP contendo informações de cabeçalho do centro de custo, incluindo descrições.
    {{ project_id_tgt }}.{{ dataset_reporting_tgt }}.CostCentersMD Tabela de destino contendo dados mestres do centro de custo, incluindo descrições em diferentes idiomas.
    {{ project_id_tgt }}.{{ dataset_reporting_tgt }}.AccountingDocuments Tabela de destino contendo dados do documento contábil, incluindo valores em moedas locais e de documento.

    Key Use Cases

    • Analisar a hierarquia e os relacionamentos do centro de custo.
    • Relatar descrições e dados financeiros do centro de custo.
    • Identificar centros de custo associados a documentos contábeis específicos.

    Query Logic

    A consulta une a tabela csks_hier com a tabela setheadert para obter informações de hierarquia e descrição do centro de custo. Em seguida, ele une a tabela resultante com a tabela CostCentersMD para adicionar dados mestres do centro de custo, incluindo descrições no idioma especificado. Finalmente, ele se junta à tabela AccountingDocuments para incluir dados do documento contábil para os centros de custo.

    Possível relação com transações SAP

    Código da Transação Descrição da Transação
    KS03 Exibir hierarquia do centro de custo
    KA03 Exibir dados mestres do centro de custo
    FBL3N Exibir itens de linha
