    AccountingDocuments

    Arquivo SQL de origem - Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - S4 - AccountingDocuments

    Visão geral

    Esta visão é uma visão de relatórios para sistemas SAP que fornece uma visão abrangente dos dados contábeis da tabela BSEG. Inclui cálculos e transformações adicionais para aprimorar os dados para fins de relatório.

    Fontes de Dados

    Tabela/Visão Descrição
    BSEG Itens da Linha do Documento Contábil
    BKPF Dados do Cabeçalho do Documento Contábil

    Key Use Cases

    • Pode ser usado para analisar dados contábeis no nível do item da linha.
    • Fornece insights sobre o desempenho financeiro de uma organização.
    • Suporta vários requisitos de relatório, como demonstrações financeiras, relatórios de gerenciamento e registros regulatórios.

    Query Logic

    A visão combina dados das tabelas BSEG e BKPF e aplica cálculos para transformar os dados para fins de relatório. Inclui lógica para lidar com vários cenários, como inverter transações de débito/crédito, calcular valores de impostos e determinar a data líquida de vencimento.

    Possível relação com transações SAP

    Código da Transação Descrição da Transação
    FB03 Exibir Documento
    FBL3N Exibição de Item de Linha
    FBL5N Visão Geral do Documento
